# QuantGuild Trading Bot

An exploratory quantitative trading bot and algorithmic framework.

## Overview

This repository holds initial stages and components of a Python-based trading bot. It aims to programmatically identify market trends, filter equity universes using structured definitions (e.g., `equities.json`), and potentially execute automated trading strategies.

## Contents
- `1. Trading Bot/`: The primary algorithmic trading implementation and backtesting module.
- `equities.json`: A predefined universe of structured equities and parameters used by the bot for scanning.

## Next Steps
This project currently serves as a scaffolding for advanced quantitative development. Expanding this repository could include API integrations with interactive brokers or cryptocurrency exchanges.
